# INFOM_DBAPP_01

Transportation Booking Management Systems

SETUP Instructions:
1. Open MySQL Workbench
2. Import transport_db_sql
3. Open the project in IntelliJ / NetBeans
4. Add mysql-connector-j-9.5.0.jar to the libraries
5. Update the DBConnection.java:
  - Username: root
  - Password: your password
6. Run the Driver.java

Features:
- Passenger Management
- Booking & Reservation
- Schedule Management
- Payment Processing
- Reports Generation
